# ❤️ Love-Sorry Website ❤️

A beautiful interactive apology website made especially for **Lovepreet**.

## ✨ Features

- 💖 Beautiful Welcome Screen
- 🌸 Animated Background
- ❤️ Click Me Button
- 🥺 Continue / Discontinue Page
- 💌 Personalized Sorry Letter
- ✨ Typewriter Animation
- 🎵 Background Music Support
- 😔 Accept My Punishment Button
- 🎲 Random Punishment Generator
- 💝 I Forgive You Button
- 😂 Funny "Not Yet" Button
- 🎉 Celebration Effect
- 📱 Mobile Friendly
- 🌙 Modern Glassmorphism UI

## 🚀 How to Run

### Option 1 (Recommended)

Open the project in **Visual Studio Code**.

Install the **Live Server** extension.

Right-click on **index.html** and select:

```
Open with Live Server
```

The website will open automatically in your browser.

---

### Option 2

Simply double-click **index.html** to open it in your browser.
